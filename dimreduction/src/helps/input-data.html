This panel allows the user to load the input data set.


Implemented Functionalities:


- (Text Field) Input File: Store the path of the input data file.


- (Button) File: Shows open dialog to select input data file. The path of the selected file is stored automatically at Input File text field.


- (Button) Read Data: Read input data file, which is specified at Input File (Text Field).


- (Button) Next Step: Move focus to the next step of the software.


- (Check Box) Are column description in first row?: If input data file has sample/feature descriptions or titles in first row, mark this check box.


- (Check Box) Are titles in first column?: If input data file has variables descriptions or titles in first column, mark this check box.


- (Check Box) Transpose the matrix?: The variables must be at rows and features/samples or time series must be at columns. Mark this check box if needed.


- (Table) Read data: Shows the read data. If your input data file has sample descriptions or titles, you can edit them directly in the table.


- (Button) Save Read Data: Click to save the read data into a text file.


Web page of DimReduction project are available at http://dimreduction.incubadora.fapesp.br.
