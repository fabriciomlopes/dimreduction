This panel allows the application of cross-validation.


Implemented Functionalities:


- (Combo Box) Criterion Function: Select the criterion function based on classifier information (mean conditional entropy) or based on classifier error (CoD - Coefficient of Determination).


- (Text Field) q-entropy: The value defined in this text field represents the type of entropy considered in criterion function. Use 1 to apply Shannon Entropy. Use a value different from 1 to apply Tsallis Entropy (smaller than 1 for subextensive entropy or larger than 1 for superextensive entropy). This parameter is used only by the criterion function based on entropy.


- (Combo Box) Penalization Method:

- no_obs: apply penalty for non-observed instances.

- poor_obs: apply penalty for poorly observed instances (observed only once).


- (Text Field) Alpha (value for penalty): Only used by no_obs penalization method. Alpha value represents the probability mass for the non-observed instances. This parameter is added to the relative frequency (number of occurrences) of all possible instances.


- (Slider Bar) Beta (value of confidence): Only used by poor_obs penalization method. Beta value is attributed to the conditional probability of the observed class, given the instance. The value (1-Beta) is equally distributed over non-observed classes of the considered instance.


- (Slider Bar) Training set size: Use the slider bar to select the training set size. The remaining data represents the test set size. The training set are randomly chosen.


- (Text Field) Number of executions: Fill this text box with an integer value, which represents the number of executions of the cross validation.


- (Button) Execute Cross Validation: Execute cross validation on input data (if input data are quantized) or on quantized data. The data must have the label of the classes in its last column.


- (Progress Bar): Display the progress of the cross validation execution.


- (Option Group) Search Method:

- SFS: Apply SFS algorithm to perform feature selection.

- SFFS: Apply SFFS algorithm to perform feature selection.

- Exhaustive Search: Apply Exhaustive algorithm for feature selection.

- (Text Field) Maximum Set Size: Select the maximum cardinality of a feature set to perform the search. Use this option only for SFFS algorithm.


- (Button) Scatter Plot: Display a scatter plot chart with two selected features indexes (columns) of the input data.


- (Button) Parallel Coordinates: Display parallel coordinates chart with selected features indexes (columns) from the input data.


- (Check Box) Average values only: Mark this check box to plot only the average values of the features (columns) in parallel coordinates chart.


- (Check Box) Normalize values: Mark this check box to plot normalized values between 0 and 1 of the features (columns) in parallel coordinates chart.


- (Button) Signal Plot: Display a signal plot of the selected variables indexes (rows) of the input data.


- (Button) Correlation Coefficient: Display a table with correlation coefficient of the selected features indexes (columns) of the input data.


- (Button) Plot Cross Validation Results: Display a chart with classification results, obtained by cross validation execution.


- (Text Area) Results:

- Classification results (right side): Shows the criterion value for the selected features.

- Feature Selection results (left side): Shows targets and its predictors or predictors and its targets, and the criterion value. If ties occur, all the tied sets are displayed.


- (Button) Save Results: Click to save results into a text file.
